# Pokémon TCG Binder Tracker

A collection of interactive HTML binder apps for tracking Pokémon TCG cards.

## Binders

### MEGA Dream ex (M2a) - High Class Pack
- Full set tracker for the Japanese High Class Pack MEGA Dream ex
- Ownership tracking with visual binder layout
- Watchlist functionality
- Copy-to-clipboard for missing cards list
- Data saved locally via localStorage

### Penny (ボタン) Character Binder
- Multi-set, multi-language character binder for Penny / ボタン
- Spans across multiple Pokémon TCG sets

## How to Use

1. Open any `.html` file directly in your browser — no server needed
2. Click cards to mark them as owned
3. Use the watchlist and missing cards features to manage your collection

## Tech

Pure HTML/CSS/JavaScript — no frameworks, no build step. Just open and go.
